Proyecto de Cuidado de Hortalizas

Visión por Computadora y Procesamiento de Datos

Introducción

En este proyecto, hemos desarrollado un sistema integral para el cuidado de hortalizas utilizando visión por computadora y procesamiento de datos. El proyecto se divide en dos partes principales:

Visión por Computadora

Responsable: Kevin

Utilizando técnicas de visión por computadora, cargamos imágenes de las plantas y aplicamos varios filtros y codificaciones de colores para analizar su estado. Este análisis permite determinar si las plantas son propensas a tener hongos, basándose en los porcentajes obtenidos.

Ver más detalles

Procesamiento de Datos

Responsable: Sebastián

Mediante el procesamiento de datos obtenidos de las hortalizas (como temperatura, humedad y pH), limpiamos y analizamos los datos para generar estadísticas útiles. Estas estadísticas nos ayudan a evaluar la calidad de los cultivos, clasificándolos en buenos, regulares o malos.

Ver más detalles

Implementación

El proyecto se implementó utilizando microcontroladores como el ESP32, Raspberry Pi 4 y un NAS para el almacenamiento y procesamiento de datos.

Desarrollo de la Página Web

Estamos creando una página web con Django que servirá como una plataforma central para explicar el proyecto en términos generales y mostrar nuestros resultados. La página incluirá secciones detalladas sobre cada parte del proyecto y permitirá a los usuarios cargar sus propios datos para evaluar la calidad de sus cultivos.
